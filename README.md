## Configuration

### Network Setup

The application is configured via the `.env` file. To switch between testnet and mainnet:

1. Open `.env` file
2. Change `NEXT_PUBLIC_NETWORK=testnet` to `NEXT_PUBLIC_NETWORK=mainnet` (or vice versa)
3. Restart your development server
